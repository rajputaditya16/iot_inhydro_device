  § 2   DATA SOURCE — ThingSpeak Field Mapping
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ThingSpeak API response (feeds array):
  ┌──────────┬──────────────────────────────────────────────────────────┐
  │  FIELD   │  MEANING                                                 │
  ├──────────┼──────────────────────────────────────────────────────────┤
  │ field1   │  Water Temperature  →  [Room1_val, Room2_val]  (°C)      │
  │ field2   │  Moisture           →  [Room1_val, Room2_val]  (%)       │
  │ field3   │  EC (mS/cm)         →  [Room1_val, Room2_val]            │
  │ field4   │  pH                 →  [Room1_val, Room2_val]            │
  │ field5   │  Room Temperature   →  [Room1_val, Room2_val]  (°C)      │
  │ field6   │  Room Humidity      →  [Room1_val, Room2_val]  (%)       │
  │ field7   │  CO2                →  [Room1_val, Room2_val]  (ppm)     │
  │ field8   │  ORP                →  [Room1_val, Room2_val]  (mV)      │
  ├──────────┼──────────────────────────────────────────────────────────┤
  │ created_at│  Timestamp of each reading (ISO 8601 format)            │
  └──────────┴──────────────────────────────────────────────────────────┘

  EC Scaling Formula applied before display:
    raw_ec  →  (raw_ec × 0.85) / 1000  =  ec_mS/cm
    ec_mS/cm × 500  =  TDS in ppm

  § 3   CSV EXPORT — How It Works
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Technique: Pure JavaScript — No API call, No backend, No extra library

  Steps:
    Step 1 ──►  feeds array (already in state) ko CSV string mein convert karo
    Step 2 ──►  JavaScript Blob object banao (type: "text/csv")
    Step 3 ──►  URL.createObjectURL(blob) se temporary URL banao
    Step 4 ──►  Ek hidden <a> tag create karke .click() trigger karo
    Step 5 ──►  Browser apne aap file download kar leta hai  ✅

  CSV Output Columns:
  ┌─────────────────────────────────────────────────────────────────────┐
  │ Timestamp | Water Temp (°C) | Moisture (%) | EC (mS/cm) | TDS (ppm)│
  │ pH | Room Temp (°C) | Room Humidity (%) | CO2 (ppm) | ORP (mV)     │
  └─────────────────────────────────────────────────────────────────────┘

  Time Range Filter Options:
    ◉  Last 24 Hours
    ◉  Last 7 Days
    ◉  Last 30 Days
    ◉  Custom Date Range  (ThingSpeak ?start=&end= params support karta hai)

  Filename format:  inhydro_data_2026-06-10.csv
